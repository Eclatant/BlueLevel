## Commit Convention
- git add -p을 통해 add
- git commit -v를 통해 commit
- 한 줄로 설명할 수 있는 단위로 나누어 커밋
- 어떻게 보다는 무엇과 왜를 설명
- 첫 줄 외에 나머지 줄에 항상 자세히 기록
- [`branch-name`] `Capital present-verb` ~ \n `description`
